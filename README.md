# p-code virtual machines experimentation sandbox



- **mk1** - as close to Wirth's machine as possible
- **mk2** - variable number of arguments, swapped a and l
- **mk3** - bytecode version of mk2



Planned extensions:

- io:
  - command-line arguments
  - getc / putc
  - geti / puti
  - file descriptors
