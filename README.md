# p-code virtual machines experimentation sandbox v2



- **mk1** - as close to Wirth's machine as possible
- **mk2** - variable number of arguments, swapped a and l
- **mk3** - internal bytecode version of mk2



Extension 1:

- INC, DEC
- AND, OR, XOR, NOT
- SHL, SHR, SAR

Extension 2:

- PUTC, PUTI
